# MechMind · מהנדס-העל ⚙️

**עומק מכני לפי דרישה — למהנדסי תעשייה וניהול ומנהלי פרויקטים.**

מתארים כוונה בשפה חופשית או מעלים תמונת שרטוט — ומקבלים תוצר הנדסי אמיתי:
מודל STEP/DXF, כתב כמויות Excel, בדיקת שפיות חוזק, המלצת חומר ותהליך ייצור,
ותוכנית פרויקט עם מפרט RFQ לספק.

> **העיקרון המרכזי:** כל מספר הנדסי (מאמץ, מקדם ביטחון, נפח, מסה, עלות) מחושב
> בקוד דטרמיניסטי — לא על-ידי ה-AI. ה-AI מפרש כוונה ומנסח; הכלים מחשבים.

## הרצה מקומית — בדיוק הפקודות האלה

### 1. Backend (טרמינל ראשון)

```bash
cd mechmind
python3.11 -m venv .venv
.venv/bin/pip install -r requirements.txt
cp .env.example .env
# ערוך את .env והדבק את מפתח ה-API שלך:
#   ANTHROPIC_API_KEY=sk-ant-...
.venv/bin/uvicorn backend.main:app --host 0.0.0.0 --port 8000
```

### 2. Frontend (טרמינל שני)

```bash
cd mechmind/frontend
npm install
npm run dev
```

### 3. פתח בדפדפן

```
http://localhost:5174
```

**בלי מפתח API?** המוצר עדיין עובד: בדיקת חוזק, יועץ חומרים ותכנון ייצור הם
דטרמיניסטיים לחלוטין וזמינים דרך ה-API. הצ'אט, קורא השרטוטים וסטודיו ה-CAD
דורשים מפתח (יוצג באנר "AI כבוי" בממשק).

### הרצת הבדיקות (72 בדיקות)

```bash
cd mechmind
.venv/bin/python -m pytest backend/tests -q
```

## קו הבטיחות 🔴

כל תוצר חוזק נושא אוטומטית (נאכף בקוד, לא תלוי במודל):

> אומדן ראשוני בלבד. אינו מחליף חתימת מהנדס רשוי. אין להשתמש בו לתכן סופי
> של ציוד נושא, הרמה, לחץ, או כל דבר שאדם עומד לידו.

- מקדם ביטחון < 2 → דגל אדום + הפניה למהנדס.
- עומס דינמי / עייפות / כלי לחץ → המערכת מסרבת לחשב ומפנה למהנדס.

## אבטחה

- סקריפטי CAD שנוצרים על-ידי ה-LLM עוברים **ולידציית AST** (רק cadquery/math,
  בלי קבצים/רשת/רפלקשן) ורצים ב-**subprocess מבודד**: סביבה נקייה, מגבלות
  CPU/זיכרון, timeout קשיח.
- הורדת קבצים מוגנת מפני path traversal — מוגשים רק קבצים מתוך תיקיית התוצרים.
- כל תוכן משתמש/LLM שנכנס ל-PDF עובר HTML escape.
- הגבלת קצב לבקשות יקרות. מפתחות רק ב-`.env` — לעולם לא בקוד.

## פריסה לענן

**Backend (Render):**
- Build: `pip install -r requirements.txt`
- Start: `uvicorn backend.main:app --host 0.0.0.0 --port $PORT`
- Environment: `ANTHROPIC_API_KEY`, `CORS_ORIGINS=https://<הדומיין-של-הפרונט>`

**Frontend (Vercel / Render Static):**
- Build: `npm run build` · Output: `dist`
- Environment: `VITE_API_URL=https://<כתובת-ה-backend>`
